Forma de la tabla

18 de Marzo 2018, Andres y Luis

Para la creacíón de la base de datos hemos utilizado el servicio de php que ofrece Wampserver, y cuenta con una sola tabla denominada "miembros" de dos columnas:

  • DNI
  • USUARIO
Cada columna, tiene la función de ser un atributo de tipo String. Además la columna o atributo DNI, es la clave primaria de la tabla para poder recorla, de manera que es importante que el valor de este atributo no se puede repetir, de lo contrario, saltaría o se produciría una error.
